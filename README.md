# 3D Portfolio

A personal portfolio website designed to present projects, skills, and experience in a modern and interactive way.

## Live Demo

https://chandan404.netlify.app/

## About

This project is built to showcase a developer portfolio with a focus on clean design, smooth interaction, and a visually engaging user experience. It is created to reflect both creativity and technical skills in a simple and professional layout.

## Features

- Modern portfolio layout
- Interactive and responsive design
- Smooth animations and transitions
- Clean project presentation
- Easy to navigate

## Technologies Used

- HTML
- CSS
- JavaScript

## Getting Started

To run this project locally:

1. Clone the repository  
2. Open the project folder  
3. Launch `index.html` in your browser
