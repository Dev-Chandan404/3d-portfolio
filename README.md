## ✨ About the Project

> A **breathtaking 3D personal portfolio** built to stand out from every flat, scrolly site on the internet. Featuring a real-time orbiting planet you can drag with your mouse, scroll-driven cosmic animations, and a futuristic glassmorphic UI — all powered by Three.js and GSAP.

This isn't just a portfolio. It's a **dimension**.

---

## 🎯 Key Features

| | Feature | Description |
|---|---|---|
| 🌍 | **Interactive 3D Planet** | Drag to orbit, scroll to explore — powered by a custom `.glb` model |
| 🌀 | **Depth & Parallax** | Full 3D scene with dynamic lighting, camera movement & floating elements |
| ⚡ | **Cosmic Smoothness** | Lenis + GSAP deliver buttery scroll through space |
| 🎨 | **Neon Glassmorphism** | Futuristic UI panels floating in the cosmic void |
| 📱 | **Responsive Across Dimensions** | Flawless from mobile screens to ultra-wide monitors |
| 🚀 | **Vite-Powered Hyperspeed** | Instant HMR and optimized production builds |

## 🚀 Getting Started

```bash
# Clone the 3D universe
git clone https://github.com/Dev-Chandan404/3d-portfolio.git
cd 3d-portfolio

# Install dependencies
npm install

# Launch dev server → http://localhost:5173
npm run dev

# Build for production
npm run build
```

## 📁 Project Structure

```

├── 📂 public/                  # Static assets (images, textures)
```
