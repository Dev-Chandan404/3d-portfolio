## ✨ Features

| | Feature | Description |
|---|---|---|
| 🌍 | **Interactive 3D Planet** | Real-time mouse drag + scroll orbit using a custom `.glb` model |
| 🌀 | **Depth & Parallax** | Full 3D scene with lighting, camera movement & floating elements |
| ⚡ | **Buttery Smooth** | Lenis scroll + GSAP animations for silky 60fps performance |
| 🎨 | **Glassmorphism UI** | Futuristic neon glassmorphic design language floating in space |
| 📱 | **Fully Responsive** | Pixel-perfect from mobile screens to ultra-wide displays |
| 🚀 | **Vite Powered** | Near-instant HMR and optimized production builds |

---

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| **Core** | React 19, Vite, JavaScript |
| **3D Engine** | Three.js, React Three Fiber, `@react-three/drei` |
| **Styling** | Tailwind CSS, Glassmorphism |
| **Animation** | GSAP, Lenis, Maath |
| **Assets** | Custom `.glb` planet model |
| **Deploy** | Netlify |

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/Dev-Chandan404/3d-portfolio.git
cd 3d-portfolio

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open `http://localhost:5173` in your browser. Drag the planet. Scroll to explore.
