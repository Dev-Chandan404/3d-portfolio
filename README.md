## ✨ About The Project

> A cutting-edge **3D interactive portfolio** that goes beyond the ordinary. Built with the latest web technologies to deliver a stunning visual experience.

This portfolio combines **Three.js 3D graphics**, **GSAP animations**, and **React's component architecture** to create an immersive journey.

## 🚀 Getting Started

### Prerequisites
- Node.js >= 18.0.0
- npm >= 9.0.0 or yarn
- Git

### Quick Start

```bash
git clone https://github.com/Dev-Chandan404/3d-portfolio.git
cd 3d-portfolio
npm install
npm run dev
