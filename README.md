## 🧠 About

This isn’t just a portfolio—it’s an **interactive cosmic journey**.  
Designed to showcase frontend development skills in the most engaging way possible, the site features:

- A fully interactive **3D planet** that responds to mouse orbit
- **GSAP‑powered scroll animations** and cinematic transitions
- A responsive, mobile‑first layout that loads fast and looks sharp on any device

The goal is simple: **to turn a developer’s story into an experience visitors will remember**—from the first pixel to the last micro‑interaction.

---

## ✨ Features

### 🌍 3D Interactive Scene
- **Three.js** renders a rotating planet with realistic lighting and shadows
- **Mouse orbit controls** let visitors explore the 3D world from any angle
- **Immersive particle fields** and morphing geometries create a dynamic background

### 🎬 Cinematic Animations
- **GSAP** powers scroll‑triggered reveals, smooth camera transitions, and buttery‑smooth UI animations
- **Framer Motion** adds polished entrance and exit effects for UI components

### 📱 Fully Responsive
- **Mobile‑first design** ensures perfect experience on all screen sizes
- **Performance optimised** (code splitting, image optimisation) for Lighthouse scores above 90

### 🧩 Modular & Customisable
- **React 19** components for clean, reusable architecture
- **Tailwind CSS** for rapid, utility‑first styling
- **Centralised content management** — update projects, skills, and contact info without touching JSX

### 📬 Functional Contact Form
- **EmailJS integration** lets visitors send messages directly from the site
- Works out of the box — no backend required
