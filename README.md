## 🚀 Getting Started

### Prerequisites

- **Node.js** >= 18.0.0
- **npm** >= 9.0.0 or **yarn**
- **Git**

### ⚡ Quick Start

```bash
# Clone the repository
git clone https://github.com/Dev-Chandan404/3d-portfolio.git

# Enter the project directory
cd 3d-portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
